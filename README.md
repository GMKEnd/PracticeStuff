## Description  
练习项目，大概想到哪就会做到哪，暂无明确规划

### Implemented  
* RecyclerView 向下滑动到末尾时自动加载新的元素，当前TextView与ImageView反复加载  
* 回到顶部可拖动式按钮  
* 悬浮窗口包含按钮添加Header与Footer，按钮删除Header与Footer;当无可删除时Toast提示  
* 悬浮窗展现出的内容位置根据悬浮按钮位置决定  
* 在顶部时下拉松手刷新  
* 导航栏，仅支持点击Tab切换
